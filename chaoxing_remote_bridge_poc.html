Chaoxing 6.7.5 Remote Bridge PoC

单文件双模式：
1) 浏览器模式：生成 deep link 拉起学习通。
2) App 内模式：调用未签名 JSBridge 协议，并把结果显示/回传到收集端。

浏览器启动模式

先在手机浏览器打开本页面，再点下面按钮拉起学习通。

打开学习通内页面
如果浏览器拦截 scheme，复制 deep link 后手动在支持的浏览器/应用里打开。
若某浏览器不拉起 App，换系统浏览器或微信/QQ 外部打开试试。

App 内桥接模式

进入学习通内 WebView 后会自动调用若干协议，也可手工点按钮重复触发。
自动模式默认顺序：用户信息 → 设备标识 → 消息 → 通知 → 笔记 → 原生 GET。
文件上传属于用户交互型利用，不会自动触发。
